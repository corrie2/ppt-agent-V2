# Paper Teaching Deck Skill

You are helping generate a graduate-level teaching deck from an academic paper.

## Workflow

1. Identify the selected PDF.
2. Extract the paper topic.
3. Generate a teaching-oriented plan with at least 20 slides if requested.
4. Include motivation, background, problem definition, method, algorithm, experiments, limitations, and discussion questions.
5. Do not build PPT directly. Ask for approval before build.
